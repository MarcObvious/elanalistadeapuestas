Nowadays there is a growing expansion of online betting sites. As a result a new figure has emerged: the ‘Tipster’. The tipster is a betting expert who advises about betting decisions, usually just based on subjective intuition.

The aim of this project is to build a fully functional platform containing reliable information about football soccer past matches as well as Machine-Learning-based predictions of future matches results. With this aim, a large amount of data has been collected and used to build a database containing all the scores -among other information- for all the matches played in three major European leagues (BBVA, Bundesliga and Premier League) from 2013 until today.

After cleaning the data and extracting relevant features from it, a predictive model based on different Machine Learning algorithms has been induced. The model is able to learn from its own errors and is updated with new data every time a match is played. The model provides predictions to an online site to be displayed along with other statistics.
